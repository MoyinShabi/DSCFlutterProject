# ToDos- DSC Flutter Project

A Simple To Do List app

###### Simple Features:

- [x] Add a ToDo

- [x] Delete a ToDo (Swipe left to Delete)

- [x] Undo a Deletion

- [x] Mark a ToDo as completed

- [ ] Edit a ToDo

- [ ] Prioritize ToDos (move them up or down)
